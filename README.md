# hoodbet.fun landing

Marketing site for [hoodbet.fun](https://hoodbet.fun) — manifesto, roadmap, HoodBet branding.

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# deploy dist/ to hoodbet.fun
```

## Environment

| Variable | Default |
|----------|---------|
| `VITE_APP_URL` | `http://localhost:5174` |
| `VITE_GITHUB_URL` | `https://github.com/hoodbet-fun` |
| `VITE_GITBOOK_URL` | `https://hoodbet-fun.gitbook.io` |
