# hoodbet.fun landing

Marketing site for [hoodbet.fun](https://hoodbet.fun) — manifesto, roadmap, HoodBet branding.

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# deploy dist/ to hoodbet.fun
```

Public URLs (`app.hoodbet.fun`, GitBook, GitHub) are hardcoded in `src/App.jsx`.
